メールを取り込んでしおりを作成

テキストを貼り付けるか、PCのGmail（ブラウザ版）で本文をコピーして「リッチ貼り付け」を使うと、図・表・画像・書式を保持して取り込めます。読んだ位置は自動保存。完成したら「📤書き出し」でOneDriveに置けばiOSでも開けます。

Gmail webで本文をコピー→押す

データはこの端末のブラウザ内(localStorage)に保存されます。AI整形にはAPIキー設定（⚙️）が必要です。

⚙️ AI整形の設定

使うAIを選び、そのAPIキーを入力すると「✨ AI整形 / しおり再生成」が使えます。キーはこの端末のブラウザ内にのみ保存されます（プロバイダごとに別保存）。

☁️ クロスデバイス同期（GitHub Gist）

GitHub Personal Access Token（gist スコープのみ）を入力すると PC↔iPhone で自動同期します。トークンはこの端末にのみ保存されます。

🟣 Obsidian Vault 保存

記事を Markdown ノートとして Obsidian Vault 内のフォルダへ保存します（PC Chrome）。保存先フォルダを一度選ぶとブラウザを再起動しても記憶されます（File System Access API）。iOS など未対応環境では `.md` ファイルのダウンロードになります。
